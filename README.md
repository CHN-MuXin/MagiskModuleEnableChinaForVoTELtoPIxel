# Magisk Pixel启用中国VoTEL 模块



## 描述

该模块通过修改 mbn_sw.txt 文件实现开启中国 VoLTE支持 以让Pixel能使用中国电信的 VoLTE通话 ，理论上支持Pixel所有型号以及安卓版本。
